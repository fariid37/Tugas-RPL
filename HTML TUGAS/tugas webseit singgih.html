HTML

1. Pengertian HTML

HTML adalah singkatan dari Hypertext Markup Language. Disebut hypertext karena di dalam HTML sebuah text biasa dapat berfungsi lain, kita dapat membuatnya jadi yang dapat

berpindah dari satu halaman ke halaman lainnya hanya dengan meng-klik text tersebut.

2. Kode HTML

Kode HTML yang kita buat akan diterjemahkan web browser agar tampil seperti yang dirancang. Browser(biasa disebut browser) adalah sebuah software aplikasi untuk menerima, menampilkan, dan menerjemahkan informasi dari world wide web.
